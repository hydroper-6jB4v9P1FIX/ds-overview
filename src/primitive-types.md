# Primitive types

## Integers

- `u8`, `i8`, `u16`, `i16`, `u32`, `i32`, `u64`, `i64`
- `bigint` — Arbitrary-range signed integer.

## Floating points

- `f32`, `f64` — IEEE 754 floating points

## String

- `str` — UTF-8 encoded string
- `char` — Unicode Scalar Value

## Other

- `()` — Unit type
- `bool`
